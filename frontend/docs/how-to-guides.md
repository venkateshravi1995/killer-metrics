# Frontend How-to Guides

## Change a tile’s data source
1. Open a tile’s configuration panel.
2. In **Data** → **Data mode**, select `timeseries` or `aggregate` when the tile supports both.
3. Update the group-by fields and metrics if the new mode restricts them.

## Customize series colors
1. Open **Visuals** for a chart tile (line/area/bar/donut).
2. Enable series color overrides.
3. Pick colors for each series (legend entry or donut segment).

## Add filters to a tile
1. Open **Data** → **Filters**.
2. Add a filter and select the dimension.
3. Choose one or more dimension values.
